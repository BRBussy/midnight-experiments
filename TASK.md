This is an experiments repository. The goal is to be able to run minimal examples testing out midnight blockchain features before moving them into other repositories. It is a place for agents and humans to try stuff out.

It needs some work to make running experiments easier:
- it doesn't properly separate different types of experiments, namely benchmarking experiments and just general testing out new feature experiments
- experiment integration tests are merged together, they should be alongside the experiment
